# Google Analytics Content Group 內容分類


## 設定方式

```javascript
// gtag.js:
gtag('config', 'GA_TRACKING_ID', {'content_group<索引號碼>': '<群組名稱>'});

// analytics.js
ga('set', 'contentGroup<索引號碼>', '<群組名稱>');

// gtag.js:
gtag('config', 'GA_TRACKING_ID', {'content_group1': '男性'});

// analytics.js:
ga('set', 'contentGroup1', '男性');
```
